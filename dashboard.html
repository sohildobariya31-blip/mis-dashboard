☁️ SharePoint / OneDrive Connection

How to get the link:
1. Open the file in SharePoint/OneDrive
2. Click Share → Copy link → set to "Anyone with the link can view"
3. Copy that link and paste below

Or use the direct download URL:
Right-click the file in SharePoint → Copy link, then change the end to &download=1

🔗 Google Sheets Connection

In Google Sheets: File → Share → Publish to web → select sheet → format: CSV → Publish → copy the URL.
Add one URL per sheet. The dashboard will fetch live data from these URLs.

Supports BOM, Other Material, or any sheet with standard purchase columns. Multiple files/sheets are merged automatically.
